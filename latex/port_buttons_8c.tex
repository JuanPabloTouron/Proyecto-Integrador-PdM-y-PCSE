Implementations of the wrapper GPIO HAL functions. 

This file contains function implementations for defining GPIO for buttons with external interrupts. It relies on the HAL functions provided by \doxylink{stm32f4xx__hal_8h}{stm32f4xx\+\_\+hal.\+h}.
